Anglin - Fishing Skill-Up Sound
================================

By default, Anglin plays "Fishing skill level increased" when your chatlog
shows a message like:

    Character's fishing skill reaches level 42.

To use your own sound instead, drop a file named exactly:

    skillup.wav

into this folder. Anglin checks for it automatically the next time you
load the addon (/lua reload anglin) - no other configuration needed.

You can turn the alert on/off (and preview it) from the addon's Settings
window, under "Fishing Skill-Up Alert".
